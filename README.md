Pengenalan 10 Tag HTML

html
Digunakan sebagai tag utama yang membungkus seluruh isi halaman website.

head
Digunakan untuk menyimpan informasi tentang website (tidak ditampilkan di halaman).

body
Digunakan untuk menampilkan semua isi website.

h1
Digunakan untuk membuat judul utama (heading terbesar).

p
Digunakan untuk membuat paragraf teks.

a href
Digunakan untuk membuat link (tautan). Saat diklik akan menuju ke halaman lain.

img
Digunakan untuk menampilkan gambar.

src = sumber gambar
alt = teks alternatif jika gambar tidak tampil

ol
Digunakan untuk membuat daftar terurut (menggunakan angka).

form
Digunakan untuk membuat form input dari pengguna.

iframe
Digunakan untuk menampilkan konten dari website lain (contohnya video YouTube).
